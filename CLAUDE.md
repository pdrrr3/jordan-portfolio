# Jordan Sowunmi Portfolio

## Project
Single-page portfolio website for Jordan Sowunmi, a brand and cultural strategist.

## Tech
- Plain HTML/CSS (no framework)
- Font: Authentic Sans Pro (woff2 in `font/` directory)
- Logos: PNG files in `logos/` directory (white on transparent)

## Structure
- `index.html` — main page
- `css/style.css` — all styles
- `logos/` — inline brand logos
- `font/` — Authentic Sans Pro font files

## Design
- Black background, white text
- Inline brand logos flow within paragraph text
- Responsive: desktop (nav has email), mobile (email moves to footer)
- Typography-driven, minimal layout

## Figma
Use the Figma MCP server to read exact design specs. Ask for the Figma file URL to inspect font sizes, spacing, margins, and layout properties directly from the source file. Match the Figma design precisely.

## Task
Refine the HTML/CSS to match the Figma design exactly — font sizes, line heights, spacing, margins, logo sizing, and responsive breakpoints.
